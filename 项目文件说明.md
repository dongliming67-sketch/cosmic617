# 📦 Cosmic拆分智能体 - 项目文件说明

## 🎯 核心启动文件

### 1. 启动.bat ⭐ **推荐使用**
- **类型**: Windows批处理脚本
- **用途**: 一键启动整个应用
- **使用**: 双击即可运行
- **优点**: 
  - ✅ 兼容性最好
  - ✅ 无需任何配置
  - ✅ 自动检查环境
  - ✅ 自动安装依赖
  - ✅ 自动打开浏览器

### 2. 启动.ps1
- **类型**: PowerShell脚本
- **用途**: 一键启动（PowerShell版本）
- **使用**: 右键 → "使用PowerShell运行"
- **优点**:
  - ✅ 更美观的彩色输出
  - ✅ 更强大的错误处理
  - ⚠️ 可能需要设置执行策略

### 3. 启动脚本使用说明.md
- **类型**: 使用文档
- **内容**: 
  - 详细的使用步骤
  - 常见问题解答

## 🚀 启动流程

### 使用一键脚本启动（推荐）

```
双击 启动.bat
    ↓
检查Node.js环境
    ↓
检查并安装依赖
    ↓
验证配置文件
    ↓
启动后端服务 (端口3001)
    ↓
启动前端服务 (端口5173)
    ↓
自动打开浏览器
    ↓
✅ 开始使用！
```

### 手动启动流程

```bash
# 1. 安装依赖（仅首次）
npm install
cd client && npm install

# 2. 启动服务
npm run dev
```

## 🔧 配置文件说明

### .env 文件
存储API配置和环境变量：

```env
# 智谱GLM API配置（免费）
OPENAI_API_KEY=你的API密钥
OPENAI_BASE_URL=https://open.bigmodel.cn/api/paas/v4
OPENAI_MODEL=glm-4-flash

# 服务器端口
PORT=3001
```

**重要提示**:
- ✅ 已配置智谱GLM免费API
- ✅ 无限tokens，永久有效
- ⚠️ 如需更换API，请修改此文件
